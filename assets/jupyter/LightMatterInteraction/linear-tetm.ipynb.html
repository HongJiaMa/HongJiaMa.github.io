1. Model

$$ -\Delta \psi_{+} + \Re[U(x,y)]\,\psi_{+} + i \Im[U(x,y)]\,\psi_{+} + \beta \left(\partial_x - i \partial_y\right)^2 \psi_{-} + \Omega\psi_+ = E\psi_{+} $$ $$ -\Delta \psi_{-} + \Re[U(x,y)]\,\psi_{-} + i \Im[U(x,y)]\,\psi_{-} + \beta \left(\partial_x + i \partial_y\right)^2 \psi_{+} - \Omega\psi_- = E\psi_{-} $$

## Define unit cell potential: Lieb lattice
Lconstant = 6.0 # lattice constant, micron
Rpillar = 1.5 # pillar radius, micron
UinmeV=0.-0.1*1j # meV
UoutmeV=10-0.5*1j # meV
E0=hbar*hbar/(2.*meff*Lconstant*Lconstant*1.e-12)/(1.6e-22) # characteristic energy, meV
U = exp2d.U_Lieb(M, Rpillar/Lconstant, UinmeV/E0, UoutmeV/E0)
